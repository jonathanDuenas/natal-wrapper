# natal-wrapper

Wrapper for the re-natal library

[![Clojars Project](https://img.shields.io/clojars/v/natal-wrapper.svg)](https://clojars.org/natal-wrapper)

## Usage
### Import library in the ns as:
```clojure
(:require
  [natal-wrapper.core as :nw])
```
